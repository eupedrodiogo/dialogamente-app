## 📋 Resumo do Projeto

## 🔧 Tecnologias Utilizadas

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + Radix UI
- **Backend**: Supabase (Database + Auth + Functions)
- **Pagamentos**: Stripe Integration
- **Deploy**: Configurado para Vercel/Netlify
